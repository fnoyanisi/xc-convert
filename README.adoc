== About xc-convert 

_xc-convert_ is a simple tool to convert Nokia OSS system generated XML files to CSV (Comma Seperated Values) files and vice versa.  

_xc-convert_ is developed by mailto:fnoyanisi@yahoo.com[Fehmi Noyan ISI]
